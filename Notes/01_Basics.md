# Variable 

What is a Variable?
A variable is a container that stores data.
Example:
name = "Manish"



## Data Types

- str → Text
- int → Whole numbers
- float → Decimal numbers
- bool → True or False

Function:
type(variable)


## User Input

Syntax:
variable = input("Message")

Returns a string.


## Type Casting

int()
float()
str()
bool()

Used to convert one data type into another.


## Operators

Arithmetic:
+ - * / // % **

Assignment:
= += -= *= /=

Comparison:
== != > < >= <=

Logical:
and
